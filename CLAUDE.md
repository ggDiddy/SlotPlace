## The Keyword Analysis System (Always Follow This)

### Step 1 — Clarify Before You Analyze

If not already provided, quickly ask (max 2 questions):

- **Target geo/market** (UK? CA? NZ? Global?) — keyword volume and competition differs massively
- **Page type** (casino review? bonus page? game guide? blog post? comparison page?)
- **Domain authority level** (new site DR 0-20 / growing DR 20-50 / established DR 50+)

If the user just wants fast recommendations, skip asking and state your assumptions inline.

---

### Step 2 — Web Search Intelligence (CRITICAL)

Before outputting any keyword recommendations, **always run web searches** to get real data:

1. Search the primary topic to see what Google ranks (SERP composition reveals intent)
2. Search competitor URLs to identify their keyword angles
3. Search for "best [topic] [geo]" and "top [topic] [year]" variants to see ranking patterns
4. Check if featured snippets, People Also Ask boxes, or local packs appear (this changes strategy)

Use what you find to ground your recommendations in reality, not guesswork.

---

### Step 3 — Keyword Categorization Framework

Organize all keywords into 4 tiers:

#### 🎯 TIER 1 — Primary Money Keywords

- High commercial intent, highest competition
- These define the page's main ranking target
- Example: "best online casino UK", "no deposit bonus casino"
- Strategy: 1 per page, used in H1, title tag, first 100 words

#### 🔍 TIER 2 — Supporting Keywords (Same Intent Cluster)

- Related terms Google treats as semantically equivalent
- Example: "top online casinos UK", "UK casino sites"
- Strategy: Use naturally throughout content, in H2s and body

#### 🌱 TIER 3 — Long-Tail Variations (Lower Difficulty, High Conversion)

- 4-6 word phrases, specific user need
- Example: "best online casino UK fast withdrawal", "UK casino no deposit bonus 2025"
- Strategy: Target via subheadings, FAQ sections, dedicated sub-pages

#### 💎 TIER 4 — Semantic/LSI/Entity Terms

- Terms that prove topical authority to Google's NLP
- Example: "wagering requirements", "RNG certified", "UKGC licensed"
- Strategy: Sprinkle throughout copy; these signal expertise

---

### Step 4 — Keyword Difficulty Assessment (iGaming Scale)

Since iGaming is one of Google's most competitive niches, adjust difficulty thresholds:

| Difficulty Level     | What It Means                                      | DR Needed to Compete |
| -------------------- | -------------------------------------------------- | -------------------- |
| 🟢 Easy (0-30)       | Low competition, often ignored long-tails          | DR 10+               |
| 🟡 Medium (31-55)    | Some competition, multiple affiliate sites ranking | DR 30+               |
| 🟠 Hard (56-75)      | Dominated by big affiliates + operators            | DR 50+               |
| 🔴 Very Hard (76-90) | Top 10 = TrustPilot, Casinomeister, AskGamblers    | DR 65+               |
| ⛔ Fortress (91-100) | Operators and mega-affiliates only                 | DR 80+               |

**Always note:** In iGaming, a "medium" KD keyword that ranks on page 2 still drives near-zero traffic. Realistic ranking windows matter.

---

### Step 5 — Search Intent Classification

Every keyword gets an intent label. This determines content type:

| Intent            | Signal Words                              | Content Type                  |
| ----------------- | ----------------------------------------- | ----------------------------- |
| **Informational** | "how to", "what is", "guide", "rules"     | Blog post, guide, FAQ         |
| **Commercial**    | "best", "top", "review", "vs", "compare"  | Review page, comparison table |
| **Transactional** | "bonus", "sign up", "claim", "free spins" | Bonus/offer page with CTA     |
| **Navigational**  | Brand name + "login", "casino"            | Brand page                    |

⚠️ **iGaming Rule**: Most high-value keywords are **commercial** intent. Never write informational content for commercial keywords — Google will not rank it for those terms.

---

### Step 6 — Competitor Keyword Gap Analysis

When user provides a URL or competitor name:

1. Web search "[competitor] site keywords" or examine what pages rank
2. Identify keywords they rank for that the user's site doesn't
3. Flag keywords where competitor content is weak (thin, outdated, missing schema)
4. Mark "quick win" opportunities — keywords on pages 2-3 with a few good backlinks could jump to page 1

## iGaming-Specific Keyword Intelligence

### High-Value Keyword Patterns (Always Check These)

- `best [casino type] [geo]` → comparison pages, always commercial
- `[casino name] review [year]` → individual reviews, navigational + commercial
- `no deposit bonus [geo]` → bonus pages, transactional goldmine
- `[game name] free` / `[game name] demo` → informational, builds topical authority
- `[game name] RTP` → informational, featured snippet opportunity
- `casino bonus without wagering` → long-tail commercial, lower comp
- `fastest paying online casino` → commercial, USP-based, less saturated
- `[payment method] casino` → e.g. "PayPal casino UK" — transactional, very high intent

### Keyword Traps to Flag

- **Operator-dominated SERPs**: Some keywords (e.g. "bet365", "Betway") are brand terms — affiliates cannot rank
- **Google's "known brand" bias**: Certain terms Google shows only operators, not affiliates — always check SERP first
- **Geo-intent mismatch**: "online casino" without geo is extremely hard and vague — always push for geo-specific variants
- **Seasonality traps**: Some bonus keywords spike in Jan/Feb and die — flag these

### Emerging Opportunities (2024-2026)

- Crash games: "best crash game casino", "aviator casino sites"
- Crypto gambling: "bitcoin casino no kyc", "crypto casino bonus"
- Live dealer specific: "live blackjack online real money UK"
- Mobile-first queries: "best casino app UK", "mobile slots real money"

## CRO Best Practices

1. **CTA buttons**: Use action language — "Claim Bonus", "Get Free Spins", "Bet Now". Never "Click Here".
2. **Bonus prominence**: Bonus amount should be the largest text element on casino cards.
3. **Trust signals**: License logos, SSL badge, payout speed always near CTAs.
4. **Urgency**: "Limited offer" labels, countdown timers where appropriate.
5. **Above the fold**: Primary CTA must be visible without scrolling on mobile.
6. **Color contrast**: CTA buttons must pass WCAG AA (4.5:1 contrast ratio minimum).
7. **Loading speed**: Inline critical CSS, lazy load images, minimize JS.

---

## SEO Standards

- Use one `<h1>` per page containing the primary keyword
- Wrap FAQs in `<details>`/`<summary>` or use `itemscope` with FAQPage schema
- Casino names in `<h3>` within list items
- `alt` text on all casino logos: `alt="[Casino Name] logo"`
- Canonical tags in `<head>`
- Meta description should include bonus amount and keyword
